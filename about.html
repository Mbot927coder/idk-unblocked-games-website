About

Static game hosting, kept simple

sharpness.math is a static launcher for verified local HTML game files. The site reads from games.js, embeds games inside the Games page viewer, and stores favorites, recents, theme, and particle settings in your browser.

The current library includes 59 working games across action, adventure, puzzle, racing, shooter, and sports. Broken entries have been removed so the catalog only points at files that actually exist in Maths stuff.

Browse Games Back to Home
  • No backend or build step is required
  • Games launch from the local Maths stuff folder
  • Favorites and recently played history stay in localStorage
  • The Games page supports search, category filters, and fullscreen play
  • The layout is designed to work on desktop and mobile

Sources and Notes

The site is intentionally lightweight and mostly relies on browser APIs plus a small amount of custom HTML, CSS, and JavaScript.

  • Fonts: Space Grotesk, Orbitron, and Inter via Google Fonts
  • Storage: localStorage for favorites, recents, theme, and particle settings
  • UI: Custom client-side HTML, CSS, and JavaScript (Note that the site was made by AI)
  • Games: Games from gn-math (probably blocked but I got it on a home computer) and Unblocked Games Doc
