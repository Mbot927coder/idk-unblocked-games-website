About

Simple file-based game hosting

This site reads from a hand-built list of local game files, keeping the setup lightweight and easy to maintain. No backend, no build step — just HTML files served directly from the Maths stuff folder.

Games are embedded in an iframe so you never leave the page. Use the sidebar category links to jump straight to the type of game you want, or use the search bar on the Games page to find something specific.
